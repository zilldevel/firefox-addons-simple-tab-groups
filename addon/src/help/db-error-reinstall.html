Database error in zilldevel-tabgroups

Unfortunately, an error occurred while reading data from the ZTG addon from the Firefox database. At the moment, the addon is stopped. To resume the add-on, do the following:

Restart Firefox Browser (for Linux and Mac OS with process termination)

If you see this window again, try reinstalling the addon:
  1. Remove ZTG Addon
  2. Restart Firefox Browser
  3. Install ZTG Addon
  4. Restore the latest ZTG addon data backup located in the Firefox downloads folder "ZTG-backups"
    You can restore backup in addon settings
